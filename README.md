## Project setup
#### Install dependencies
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
Visit URL: http://localhost:8080
```

#### Compiles and minifies for production
```
npm run build
```

#### Lints and fixes files
```
npm run lint
```
